# Spotify Power BI Dashboard

## Overview

This project is an interactive Power BI dashboard built using the Spotify Top 50 dataset to analyze music trends, artist performance, and song popularity. The objective is to transform raw data into meaningful insights through clear and effective visualizations.

## Key Metrics

* Total Songs Analyzed: 789
* Unique Artists: 342
* Average Popularity: 89.62
* Average Duration: 3.28 minutes

## Features

* Artist-wise song distribution
* Analysis of most popular songs
* Monthly trends in song releases and popularity
* Album type breakdown (Single, Album, Compilation)
* Explicit vs Non-explicit content comparison

## Tools and Technologies

* Power BI
* DAX (Data Analysis Expressions)
* Data Cleaning and Transformation
